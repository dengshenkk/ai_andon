需配置三色安灯（Andon），用于ai展示运行状态（运行/告警/故障）

# 运行环境
macos12

# 核心逻辑
核心逻辑很简单，就三件事：

  1. 检测状态

  读取 ~/.claude/sessions/*.json 文件，解析 status 字段：

  "busy"  → Claude 正在工作
  "idle"  → Claude 在等你输入
  inactive → Claude 会话断开
   无文件   → Claude 没在运行

  2. 映射为颜色

  把四种状态映射到红绿灯颜色：

  ┌───────────┬─────────────┬──────────────┐
  │   状态    │   窗口灯    │  菜单栏图标  │
  ├───────────┼─────────────┼──────────────┤
  │ busy      │ 🟢 绿灯常亮 │ 绿色圆点     │
  ├───────────┼─────────────┼──────────────┤
  │ idle      │ 🟡 黄灯闪烁 │ 黄灰交替闪烁 │
  ├───────────┼─────────────┼──────────────┤
  │ idle      │ 🔴 红灯闪烁 │ 红灰交替闪烁 │
  ├───────────┼─────────────┼──────────────┤
  │ 无session │ ⚫ 全灰     │ 灰色圆点     │
  └───────────┴─────────────┴──────────────┘

  3. 循环刷新

  每帧（约 60fps）重复：检测 → 更新颜色 → 重绘，所以状态变化是实时的。

  整个应用就这一个循环，没有网络请求、没有后台服务、没有数据库，纯粹读文件 + 画灯。
